# Webium

Browser-grade DOM/CSS/JS engine for Unity UI — run standard web code natively in Unity via PuerTS.

## Installation

Add the package to your Unity project via `Packages/manifest.json`:

```json
"com.webium.core": "https://github.com/webium/webium.git"
```

## Usage

Webium lets you build 2D UI using standard HTML, CSS, and JavaScript inside Unity. It maintains a virtual DOM in C#, exposes browser APIs via PuerTS, and renders everything as native Unity UI components.
